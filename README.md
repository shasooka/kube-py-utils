# How to Run

**1. Clone the repository**

```bash
git https://github.com/shasooka/kube-py-utils.git
```

**2. Make help to run the project**
```bash
make
```

**3. pre-requiste**
```bash
    connect to k8s cluster
    Eg: kubectl config use-context minikube
```
